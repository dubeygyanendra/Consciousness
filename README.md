## Overview

Consciousness is one of the deepest unsolved problems in science and philosophy. Over the past three decades, multiple rigorous theories have been proposed — each with its own mathematical formalism, empirical predictions, and philosophical commitments. This repository collects **11 comprehensive guides** (totaling ~294 pages) that cover the full landscape:

- **9 major theories** of consciousness, each with full mathematical treatment
- **1 introductory volume** covering foundations, problems, methods, and history
- **1 synthesis volume** with cross-theory comparison tables, adversarial collaboration results, and open frontiers

All documents use consistent professional LaTeX formatting with:
- Hyperlinked tables of contents
- Color-coded tcolorbox environments (definitions, principles, mechanisms, predictions, objections, math)
- TikZ diagrams and formal equations
- Comparison tables against all other theories
- Extensive reference lists

## Reading Paths

The volumes are self-contained and can be read in any order. Here are some recommended paths depending on your background and interests:

### 🔬 Path 1: Neuroscience First
> Introduction → GWT → RPT → IIT → CEMI → HOT → AST → PP → Orch OR → Panpsychism → Synthesis

Best if you have a neuroscience background and want to start with the most empirically grounded theories before moving to more speculative ones.

### 🧭 Path 2: Philosophy First
> Introduction → Panpsychism → IIT → HOT → AST → GWT → RPT → PP → CEMI → Orch OR → Synthesis

Best if you have a philosophy background and want to start with the metaphysical foundations before diving into neural mechanisms.

### ⚔️ Path 3: Adversarial Collaborations
> Introduction → IIT → GWT → RPT → HOT → Synthesis → (remaining volumes)

Focuses on the four theories currently being tested head-to-head in the Templeton-funded adversarial collaborations.

### 🗺️ Path 4: Overview Then Deep-Dive
> Introduction → Synthesis → (whichever individual volumes interest you)

Get the big picture first, then go deep on specific theories.

### 🔢 Path 5: Math-Heavy
> IIT → PP → Orch OR → CEMI → Panpsychism → (others)

For those who want the most mathematically rigorous content first.

## What Each Document Covers

Every theory guide follows a consistent structure:

1. **Introduction** — Core claim, historical context, motivating problem
2. **Prerequisites** — Mathematical and scientific background specific to the theory
3. **Core Theory** — Central mechanism with full formal/mathematical treatment
4. **Neural Implementation** — Mapping to brain anatomy and physiology
5. **Experimental Evidence** — Supporting and challenging data
6. **Predictions** — Specific, testable predictions
7. **Comparison with Other Theories** — Systematic comparison tables
8. **Criticisms and Open Questions** — Major objections with responses
9. **References** — Core publications for further reading

### Mathematical Content Includes:
- **IIT:** Φ computation, transition probability matrices, earth mover's distance, cause-effect structures, exclusion algorithms
- **GWT:** Wilson-Cowan equations, ignition bifurcation dynamics, signal detection theory
- **RPT:** Laminar circuit models, feedforward vs. recurrent computation, mutual information
- **HOT:** Signal detection framework (d', meta-d', M-ratio), Bayesian metacognition
- **AST:** Biased competition equations, control theory (state-space, Kalman filter), attention dynamics (τ da/dt equations), dimensionality reduction
- **PP:** Variational free energy (full derivation), hierarchical generative models, predictive coding update equations, precision-weighted prediction errors, Markov blankets, active inference, expected free energy, DCM, generalized coordinates, path integrals
- **Orch OR:** Schrödinger equation, density matrices, decoherence formalism, Diósi-Penrose gravitational self-energy integral, objective reduction timescale, Gödel's incompleteness theorems, spin network area quantization, twistor incidence relations
- **Panpsychism:** Formal structure-intrinsic nature distinction, combination problem formalization, category-theoretic colimits, information-theoretic panpsychism, double-aspect information
- **CEMI:** Maxwell's equations, current dipole model, volume conductor (Poisson equation), field energy density, phase-locking value, spectral coherence, Jansen-Rit neural mass model with ephaptic feedback

## Prerequisites

These guides assume:
- **Neuroscience:** Neurons, action potentials, synapses, brain anatomy (cortex, thalamus, brainstem), EEG/fMRI at a conceptual level
- **Mathematics:** Calculus, linear algebra, probability theory, information theory at an undergraduate level
- **Philosophy of Mind:** Basic acquaintance with the mind-body problem, physicalism, dualism, functionalism (Volume 0 provides this grounding)

More advanced mathematics (variational calculus, quantum mechanics, category theory, etc.) is introduced within the relevant guides as needed.
